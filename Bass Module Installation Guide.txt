Welcome to the installation guide of Bass Module!

Everything has been made extremely simple for you to install. You must simply run the OCRP2_Install_Bass.bat file, to install the bass module.
Everything will automatically be installed to the correct locations.

*IMPORTANT* - Make sure you do not have Garry's Mod open when running the installation file. If you do, it will not succed - *IMPORTANT*
Now enjoy the music in Orange Cosmos RP 2!
